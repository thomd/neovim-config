# Neovim Configuration

My personal [Neovim](https://neovim.io/) configuration.

## Setup

```sh
git clone https://github.com/thomd/neovim-config.git ~/.config/nvim
```

## Plugins

- [multiple-cursors.nvim](https://github.com/brenton-leighton/multiple-cursors.nvim): `lua/plugins/coding.lua`
